# 601 Inc. — Style Reference
> Architectural film canvas

**Theme:** dark
**Source:** styles.refero.design

601 Inc. wields a cinematic utilitarian aesthetic — stark contrasts, structured architectural presentation. Dark canvas + single muted golden accent. Dramatic typography scaling, unornamented UI, framed cinematic imagery.

## Tokens (CSS Custom Properties)

```css
:root {
  --color-midnight-matte: #090909;
```

## Do's
- Midnight Matte (#090909) as base canvas
- Greyscale Concrete (#4f4d3c) for cards/sections
- Aged Gold (#ece4b4) reserved for display, borders, interactive
- changeling-neo weight 400, -0.04em letter-spacing always
- 4px grid; prefer 9px element gaps, 21-30px paddings
- 11.9952px radius on rounded elements
- Cinematic framed imagery, contained never full-bleed

## Don'ts
- No additional chromatic colors (only the 3-color palette)
- No thin/light font weights
- No deep shadows / heavy elevation — flat planes
- No generic sans-serif for primary headings
- Don't break -0.04em tracking
- No full-bleed imagery
