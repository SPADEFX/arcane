# Family — Style Reference
> Pixar storyboard on cream paper — wobbly blob characters in fintech.

**Theme:** light
**Source:** styles.refero.design

## Tokens

```css
:root {
  --cream-canvas: #fbfaf9;       /* page bg, warm off-white */
  --warm-stone: #f2f0ed;         /* card inset border */
  --ink: #121212;                /* near-black, single dark moment */
  --slate: #6b6b6b;              /* secondary body */

  /* Vivid primaries — for illustration fills only */
  --electric-orange: #ff6a3d;
  --grass-green: #4caf50;
  --sky-blue: #4a90e2;
  --bright-yellow: #ffc83d;
  --bubblegum-pink: #ff8fa8;

  /* Custom 'Family' display sub-> Fraunces/PP Editorial */
  --font-display: 'Fraunces Variable', 'Instrument Serif', ui-serif, Georgia, serif;
  --font-ui: 'Inter Variable', ui-sans-serif, system-ui;
}
```

## Hard rules
- Cream #fbfaf9 canvas — never pure white
- Display = custom typeface at 68px+ with -0.031em tracking (sub Fraunces)
- ONE dark CTA #121212 only — no other dark surfaces
- Cards: inset 1px shadow #f2f0ed — NEVER drop shadows
- Vivid primaries used only on illustrations (wobbly blobs with stick legs + eyes)
- Layouts: spacious, airy, illustrative — playful not utilitarian
- Body in Inter at modest sizes; display does the heavy lifting
- Illustrations are the identity, not the color palette alone
